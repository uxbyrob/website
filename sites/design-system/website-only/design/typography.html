Typography

The type in this system responds to 3 break points optimizing both readability and screen real estate.

 

Note: Web fonts are not allowed due to client security issues

 

 

Font Family: Arial

Font Weights: Regular and Bold

Line Height: 130%

 

Default text color: @gray-darker

Link color: #0170cb

Link hover decoration: Underline

Headline Color: @IHS-Green

Visited link Color: #96157C

 

Breakpoints

Small screens: 1024px X 768px to 1280px X 800px

NOTE: At this screen size 12px @brand-primary is not 508 AA compliant

 

H1 22px / 130% (28.6)

H2 18px / 130% (23.4)
H3 16px / 130% (20.8)

H4  14px / 130% (18.2)

P 12px / 130% (15.6)

SM 10px / 130% (13)

XSM 10px / 130% (13)

 

 

Paragraph breaks are 6px

Nulla vitae elit libero
a pharetra augue.

Cum sociis natoque
penatibus et magnis

 

 

Medium screens: 1281px X 801px to 1600px X 1024px

 

H1 24px / 130% (31.2)

H2 20px / 130% (26)

H3 18px / 130% (23.4)

H4  16px / 130% (20.8)

P 14px / 130% (18.2)

SM 12px / 130% (15.6)

XSM 10px / 130% (13)

 

P Space after is set to 6px to create this effect:

Nulla vitae elit libero
a pharetra augue.

Cum sociis natoque
penatibus et magnis

 

Large screens: 1601 and higher

 

H1 28px / 130% (36.4)

H2 24px / 130% (31.2)

H3 20px / 130% (26)

H4  18px / 130% (23.4)

P 16px / 130% (20.8)

SM 14px / 130% (36.4)

XSM 12px /130% (15.6)

 

P Space after is set to 6px to create this effect:

Nulla vitae elit libero
a pharetra augue.

Cum sociis natoque
penatibus et magnis

 

 

 

Tip: Line length

For maximum readability, the optimal number of characters per line is 55 to 75 but between 75 and 100 characters per line is acceptable.
