Problem

Solution

User has difficulty knowing when they leave a page/dashboard, etc knowing where they are going to land next. User is having difficulty navigating from a page and selecting other areas in Connect to visit.

UX team is recommending to integrate a filtering and navigation process utilizing the large list selector. This process will provide user a single item to select to jump to another page, displaying sections showing where the page lives. This solution also provides for multiple selection in the large list selector. The last component to this is always giving option in the fly out menu from search bar to search entire site or browse a full list.
