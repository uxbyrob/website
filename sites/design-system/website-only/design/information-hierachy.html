Information hierarchy

An information hierarchy must be setup on each page giving the user the ability to quickly evaluate the page structure. Creating one top priority with sub priorities for a page is essential. When designing, ask yourself why is the user on this page.

 

A quick test for page hierarchy is to have a user look at your proposed design for 5 seconds. Then ask what the purpose of the page is. If they can't answer that, you have more work to do.

 

 

Design Tools for setting up hierarchy:

1) In most cases users follow an "F" pattern when looking at a page for the first time. Placing important information within this natural path will help get it noticed.

 

2) Use images/icons, contrast, scale and color to give elements on the page more weight.
