My Connect Dashboard

What can user do from the dashboard?


Alerts/Notifications:

View total counts, set timezone, set daily time of delivery, set day of week for weekly, block hours of delivery. Turn off daily alerts, click links to see details of the notifications

Article Links/Documents:

View total count, click link to visit configure/edit area

Saved Searches:

View total count, click link to visit configure/edit area

Geo Locations and Saved Regions:

View total count, click link to visit list and configure/edit area for either

Saved Data Series/Charts/Queries:

View total count, click link to visit list and configure/edit area

Activity Stream:

View log by today, tomorrow, this week, last week, this month, last month, this year. This can be collapsable for easier viewing and real estate. Each entry is represented by a icon type

This page does not exist today. UX team recommends creating a high level aggragate view of a users account for all "saved" content and also a detailed log of the history of users actions in Connect as they visit the platform for a time period (up to a year?)


Dashboard can also serve as navigation for flow to inner My Connect pages/views

Goal is for user to understand what each of the sections do,


Will also throw in an 'i" icon that allows the user to click to see a tooltip revealing the purpose or functon of each section, etc.
