Icons

Icons are separated into a few categories:

Core UI Controls - These icons are used by the design system and have specific meaning and usage defined.

Contextual - These icons are used by the add-ons and have specific meanings but can be used where ever the same icon concept needed.

 

 

Coming soon:

Common icons - These icons represent common concepts that are found across applications. The intended use is documented but some latitude is allowed.

Tool bar icons - Icons commonly found together are found here. Example: Rich text editor icons

App specific icons - Icons specific to a single or family of applications
