Quick Start

Quick tips for getting up and running with CPL3.

Information

  1. The CssPatternLibrary 3 design system = CPL3 in much of the documentation on this site.
  2. CPL3 is available via an internal npm service that is only available to employees of IHSM.
    Thus, you must configure your computer to look for this internal npm service when trying to install CPL3 and other internal npm packages. Please refer to step one below.
  3. Usage: Initial downloads and subsequent updates of CPL3 are available via the use of npm.js, Yarn, and other similar tools.
    Note: As with all npm packages - updates are available on an "opt-in" basis. This means developers or build tools will need to periodically check for updates and apply them to your product for it to receive the latest updates from CPL3.
  4. Suggest changes and improvements: Coming soon

Dependencies

  1. Node.js
  2. npm.js, Yarn, or similar tools that can make use of an npm package
  3. Twitter Bootstrap
    1. jQuery

Steps

  1. Assure you are correctly set up to use the IHSM internal npm server.
    You must be connected to the IHSM network (possibly via VPN).
    Add a .npmrc (i.e. npm resource) file to your project's root folder (usually the one that has a node_modules folder) that has the following content: 
    always-auth=true
registry=http://npm-na.ihs.com/repository/public-npm-all/
    See our detailed instructions if you encounter difficulties.
  2. Run the install command 
    npm install csspatternlibrary3 --save
     from the command line of your project root folder (or wherever you are currently storing your "node_modules" folder).
  3. Include the needed CPL3 resources in your product.
    The main CSS file (which already includes all needed Twitter Bootstrap styles) 
    <link rel="stylesheet" href="css/styles-nw-min.css" />
    Some components will require the inclusion of their corresponding JavaScript files to function correctly. For example the Carousel component requires 
    <-- Either the bootstrap.min.js file which includes all of the JavaScript for all of the CPL3 components bundled together -->
<script src="js/bootstrap.min.js"></script>

<-- Or the individual carousel.js if it makes more sense to cherry-pick out only what you need. -->
<script src="js/carousel.js"></script>
