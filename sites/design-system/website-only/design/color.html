Specific use Gray

These grays have specific uses and should not be used for general UI elements. Gray-bg should be used for the backgrounds of spaces only. This includes well, panels, and pages. Gray-action is used for clickable icons only and is a second option when the color brand-action creates a hierarchy issue. An example of this is a row of icons like found in a tool bar.

Color uses

The IHSM-action color is used to communicate that an object is clickable. This color is also used to identity the active item in a list group.

The IHSM-attention color can be paired with the IHSM-action color to draw attention to an active element. For example, tabs and toolbars use a attention bar to indicate the active element.

The hover effect of all elements is achieved by darkening the original color by 12%.

Disabled

A disabled effect is created by swapping the color to @gray-light
