Component parts

Component parts are used to create as much of the UI as possible. By using these parts constancy in increased across the enterprise and future updates can be made on a global scale.

Caret

Carets are used to indicate access to additional information found in dropdown menus, tree structures and accordion headers. In the closed state carets point right and in the open state it points down to the content.

Close

Use for closing panels, modals or other spaces and containers. Close can also be used to close a tag.

Clear

The clear icon appears once a user has input data into a field either by typing or taking an action that fills a form field

Chevron

Chevrons are not interchangeable with the caret. Chevrons indicate direction of an operation like moving though a set of items. Chevrons do not rotate.

 

Menu

The "hamburger" menu is use to open a drawer. A drawer is defined as a place a user looks for an action. A Drawer is different from a shade or panel which the system produced to great more workspace based on an action taken by the user.

List Group

List groups should be used anytime a vertical list of items is required. Not all list groups need hover and selection functionality and can be used without those classes tuned on. List groups are available in 3 sizes

 

Rail Control

A double chevron is used to collapse or open a rail or panel.

Resize

Used as an affordance in the lower right corner of a modal or text box to change the size in both X and Y space

Drag

Used as an affordance to change the size of a space by dragging it from behind an object/off screen in a single X or Y direction.

Dividers

Dividers are used to separate items and spaces. Dividers can break insets to create full width spaces within an object if desired.

Title Dividers

Title Dividers are used to create a top to a space and are always accompanied by a headline or subhead.

Headline

Dropdowns

Dropdowns and Combo box selectors are not the same. Dropdowns are used to navigate and are not a replacement for a selection menu. Dropdowns are list groups in a container with shadow.

Panels

Panels are one of the most important components. Panels are used to create may other parts and patters. Border color, corner radius and padding are all built into this part which creates far reaching constancy across the system. Use insets to create consistent padding

 

Panel headings can be added to create cards. The headings have a default white background but a contextual color can be added for more emphases.

 

Shadows can be added to panels to indicate depth in the UI. See layers for more information.

Panel

Wells

Wells are used to create a grouping of related items. Shadows should never be added to wells. Wells have a standard 2px radius corners but can also come as edgeless which allow them to bleed to the edge of a space.
